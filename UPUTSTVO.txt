BG AUTO SPA — SAJT (ažurirano)

Fajlovi:
  index.html    = glavni sajt
  admin.html    = admin panel (upiti)
  config.js            = TELEFON, WhatsApp, adresa, recenzije
  supabase-config.js   = Supabase URL + anon key
  supabase-schema.sql  = SQL za bazu (jednom u Supabase)
  site.js / admin.js   = logika (ne diraj osim ako znaš šta radiš)
  favicon.svg   = ikonica u tabu
  logo.png      = logo
  gallery/      = tvoje slike (hero.jpg, about.jpg, 1.jpg–6.jpg)

VAŽNO — pre deploya na Render:
  1. Otvori config.js
  2. Zameni phone i whatsapp pravim brojevima
  3. Zameni addressDetail tačnom adresom
  4. Dodaj slike u gallery/ (vidi gallery/README.txt)
  5. Po želji izmeni recenzije u config.js (reviews)

Kako se otvara lokalno:
  - Dupli klik na index.html ili Live Server u VS Code/Cursor

Admin panel + Supabase:
  1. Otvori supabase.com → novi projekat (ili postojeći)
  2. SQL Editor → nalepi ceo fajl supabase-schema.sql → Run
  3. Project Settings → API → kopiraj URL i anon public key u supabase-config.js
  4. Authentication → Users → Add user (email + lozinka za admin)
  5. Otvori admin.html → uloguj se tim nalogom
  - Upiti sa sajta idu u Supabase (vidiš ih sa telefona i laptopa)
  - Bez supabase-config.js radi samo localStorage (isti browser)

Forma na sajtu:
  - Čuva upit u admin panel
  - Otvara WhatsApp sa popunjenom porukom

Recenzije:
  - Klijent: meni Recenzije → forma "Ostavi recenziju"
  - Admin: admin.html → Recenzije → Odobri / Odbij
  - Na sajtu se vide samo ODOBRENE recenzije
  - Prve recenzije iz config.js su učitane jednom kao objavljene

Hosting (Render):
  - Uploaduj ceo folder "detajling deki"
  - Za svoj domen: poveži DNS na Render i promeni siteUrl u config.js

Fajlovi za Supabase:
  supabase-config.js   = URL + anon key (obavezno za online)
  supabase-schema.sql  = tabele u bazi (pokreni jednom)
  supabase-db.js       = ne diraj
  admin.js             = ne diraj
